Implementing and applying numerical methods

Basic info Link to heading

  • Type: Undergraduate project - Numerical Methods course
  • Period: 2st semester of 2022
  • Authors:
    • Student: Gabriel Eduardo Lima
    • Professor: Eder Augusto Penharbel

About Link to heading

Study and implementation of different numerical methods throughout the mentioned discipline. As a highlight, it is possible to mention the implementation of:

  • Different Gauss-Jordan elimination method variations, as well as the Cholesky algorithm for solving linear systems;
  • Different interpolation methods - Nearest Neighbour, linear and bilinear - and application in image transformation;
  • Root finding algorithms;
  • CORDIC algorithm using Taylor polynomials for trigonometric functions approximation;
  • SIFT algorithm - Although not finalized, it was possible to study concepts such as the DOG calculation, and quadratic function interpolation using Taylor.

As a specification, Python and C++ languages were used to develop the codes.

As a main result, it was possible to study and learn about a set of numerical tools for problem solving, as well as to consolidate important mathematical concepts.

Content and topics studied Link to heading

Many topics have already been mentioned in the previous section, but as a bonus it’s possible to mention:

  • Numerical errors
  • Floating point arithmetic
  • Limits for data representation
  • Image processing concepts
  • Data structures for numerical methods
