Drone path programming using images and python

Basic info Link to heading

  • Type: High school project (technician course)
  • Period: Year 2019
  • Area: Image processing
  • Authors: Gabriel Eduardo Lima and Thiago Vinicius de Oliveira Hornburg
  • Instructors: Hylson Vescovi Netto and Deivis Elton Schlickmann Frainer

About Link to heading

The project consists on applying computing and mathematics knowledge in order to create automated trajectories for drones. Image analyzis and interpretation is done by software implemented in Python (3.7.0) with the Pillow library (5.4.1) to produce commands that are loaded in a Tello drone (produced by ryzerrobotics). The objective is to present a initial and simple theoretical/practical basis for drone movement automation aimed to perform some challenging tasks, such as painting high places and of great risk for a person.

As conclusion, it was possible to interpret simple padronized images (as shown in the gallery bellow) and trace the drone route. Although some problems where observed - as what happens when the image contains intersecting paths -, the project helped developing some basic concepts of image processing and programming skills.
