A differential robot simulator prototype with OpenGL

Basic info Link to heading

  • Type: Undergraduate project - Computer Graphics course
  • Period: 1st semester of 2023
  • Authors:
    • Student: Gabriel Eduardo Lima
    • Professor: Eder Augusto Penharbel

About Link to heading

The project consists in implementating basic elements necessary for a differential robot simulation in OpenGL (core). The objective, in addition to enabling the study of future work, is to apply computer graphics concepts in the C++ language in a practical way. In the end, it was possible to develop a simulation prototype. In addition, it was possible to deeply study different subjects, as 3D computing and the use of Bézier curves. Alongside this, topics such as mathematical optimization and numerical root finding problems were worked on to solve curve rendering.

Content and topics studied Link to heading

  • Graphics development using C++ and OpenGL
  • OpenGL graphics pipeline
  • Transformations
  • Bézier curves and parametric functions
  • Quadratic and cubic Bézier curve rendering
  • Numerical optimization
  • Newton’s method for root finding
  • Kinematics Differential robot Kinematics in space
  • 3D computer graphics
