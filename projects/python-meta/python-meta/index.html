Python data type validation using metaprogramming

Basic info Link to heading

  • Type: Undergraduate project - Object Oriented Programming II course
  • Period: 1st semester of 2021
  • Authors:
    • Student: Gabriel Eduardo Lima
    • Professor: Hylson Vescovi Netto

About Link to heading

The project consists of verifying and validating data types at runtime of Python programs using metaprogramming. In this way, it is possible to prevent incorrect data types from being passed to functions and class methods. The language version used was 3.8.5, and as a main resource it is possible to highlight the use of annotations. As a main conclusion, it was possible to define (and implement) an initial model for validating simple types dynamically in the language, which can be used for the development of future works.

Content and topics studied Link to heading

  • Object Oriented Programming
  • Metaprogramming
  • Annotations in Python
  • Types of data
  • Dynamic and static typing
