DRØMME: A 2D shooting game proposal

Basic info Link to heading

  • Type: High school project (technician course)
  • Period: Year 2018
  • Area: Game development
  • Authors: Gabriel Eduardo Lima, Guilherme Eduardo Kuglin, Thiago Vinicius de Oliveira Hornburg e Timóteo Rafael Janzen
  • Instructors: Adriano Pessini, Eder Augusto Penharbel e Ricardo de la Rocha Ladeira

About Link to heading

DRØMME consists of a simple 2D shooting game. It is set in a post-zombie apocalypse world, where the player must face and eliminate all enemies to progress through the stages. Along the journey, new weapons and characters can be purchased in the game internal shop. From a projetc perspective, the objective is to develop a game and discuss the processes and techniques necessary for its completion. Among the tools used, it is possible to mention the Python programming language (3.6.5) together with the pygame library (1.9.4).

Among the conclusions obtained, it is possible to say the implementation of a game is a process that requires different knowledge - programming, mathematics, software engineering, etc. Although games are usually associated with fun, the steps involved in their development can be complex. Finally, the project realization contributed to achieve familiarity and maturity in programming.

Content and topics studied Link to heading

  • Object Oriented Programming
  • Game development using pygame
  • Math for game development
  • Simple game mechanincs – Collision system – Reward system – Moving and Camera model for 2D
  • Interface design
  • Documentation and Game Design Document (GDD) writing
