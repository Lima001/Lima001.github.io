A P2P file sharing prototype system: Sending secret messages through images

Basic info Link to heading

  • Type: Undergraduate project - Web Development II course
  • Period: 1st semester of 2021
  • Authors:
    • Student: Gabriel Eduardo Lima
    • Professor: Hylson Vescovi Netto

About Link to heading

The project consists of implementing a network for sharing “secret” messages through steganographed images with an encrypted text message. A prototype of a P2P network was implemented for file access and distribution. For this purpose, an HTML page server and a P2P network prototype to operate as a file server were implemented in Python. Despite some problems, it was possible to observe the storage, sharing and retrieval of messages hidden in images as a result.

Content and topics studied Link to heading

  • Web systems architectures
  • Client-server architecture
  • P2P Network
  • Pipeline development
  • Web page development
  • Encryption concepts and techniques
    • Caesar’s cipher
    • Text steganography in images
