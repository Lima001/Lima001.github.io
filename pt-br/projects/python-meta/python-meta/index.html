Validação de tipos de dados em python usando metaprogramação

Informações básicas Link para o cabeçalho

  • Tipo: Projeto da graduação - disciplina de Programação Orientada a Objetos II
  • Período de realização: 1º semestre de 2021
  • Autores:
    • Estudante: Gabriel Eduardo Lima
    • Professor: Hylson Vescovi Netto

Sobre Link para o cabeçalho

O projeto consiste na verificação e validação dos tipos de dados em tempo de execução de programas Python usando metaprogramação. Dessa forma, é possível impedir que tipos de dados incorretos sejam repassados á funções e métodos de uma classe. A versão da linguagem utilizada foi a 3.8.5, e como principal recurso é possível destacar o uso de annotations. Como principal conclusão, foi possível definir (e implementar) um modelo inicial para a validação de tipos simples de forma dinâmica na linguagem, podendo ser utilziado para o desenvolvimento de novos trabalhos.
