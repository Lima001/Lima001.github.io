Sobre Link para o cabeçalho

O projeto em questão foi apresentado em seminário sobre o tema de Resolução de Problemas de Busca. O objetivo consiste em apresentar problemas de busca adversária, bem como apresentar um algoritmo simples de forma prática. Para isso, foi implementado o algoritmo de minmax para jogar o jogo da velha. Além do algoritmo em si, uma implementação gráfica do jogo foi produzida de forma a permitir que um usuário dispute contra a máquina. A título de especificação, foi utilizada a linguagem Python e a biblioteca Pygame.
