Implementando e aplicando um modelo MLP

Informações básicas Link para o cabeçalho

  • Período de realização: 1º semestre de 2023
  • Autores:
    • Estudante: Gabriel Eduardo Lima
    • Professor: Eder Augusto Penharbel

Sobre Link para o cabeçalho

O projeto consiste no estudo do modelo MLP e do algoritmo backpropagation. Além da exploração formal, uma implementação própria em Python foi realizada. Essa implementação foi então aplicada para a resolução de problemas como: Mapeamento da função XOR (Ou exclusivo); Classificação de dígitos do conjunto MNIST; Construção de modelo autoencoder puramente baseado em MLP para compressão de imagens do conjunto MNIST; Construção de modelo autoencoder puramente baseado em MLP para denoising de blur Gaussiano de imagens do conjunto MNIST.
