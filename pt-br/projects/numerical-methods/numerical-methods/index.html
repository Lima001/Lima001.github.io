Sobre Link para o cabeçalho

Estudo e implementação de diferentes métodos numéricos ao longo da realização da disciplina mencionada. Como destaque é possível mencionar a implementação de:

  • Diferentes variações do método de eliminação de Gauss-Jordan, bem como o algoritmo de Cholesky para solução de sistemas lineares;
  • Diferentes métodos de interpolação - Nearest Neighbour, linear e bi-linear - e aplicação para construção de imagens transformadas;
  • Algoritmo para encontrar raízes;
  • Algoritmo CORDIC em conjunto de polinômios de Taylor para aproximação de funções trigonométricas;
  • Algoritmo SIFT - Apesar de não finalizado, foi possível estudar conceitos como o cálculo da diferença de gaussianas, e a interpolação de uma função quadrática usando Taylor.

A título de especificação, foram utilizadas as linguagens Python e C++ para o desenvolvimento dos códigos.

Como principal resultado foi possível estudar e conhecer um conjunto de ferramentas numéricas para resolução de problemas, bem como consolidar conceitos matemáticos importantes.
