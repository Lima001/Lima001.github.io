Programação de caminho para drone usando imagens e python

Informações básicas Link para o cabeçalho

  • Tipo: Projeto de ensino médio
  • Período de realização: Ano de 2019
  • Autores:
    • Estudantes: Gabriel Eduardo Lima e Thiago Vinicius de Oliveira Hornburg
    • Professores: Hylson Vescovi Netto and Deivis Elton Schlickmann Frainer

Sobre Link para o cabeçalho

O projeto consiste em aplicar conhecimentos de computação e matemática para criar trajetórias automatizadas para drones. A análise e interpretação das imagens é feita por um software implementado em Python (3.7.0) com a biblioteca Pillow (5.4.1) para produzir comandos que são carregados em um drone Tello (produzido pela ryzerrobotics). O objetivo é apresentar uma base teórica/prática inicial e simples para a automação do movimento de drones visando a execução de algumas tarefas desafiadoras, como pintar locais altos e de grande risco para uma pessoa. Como resultado, foi possível interpretar imagens padronizadas simples (como mostra a galeria abaixo) e traçar a rota do drone. Apesar de alguns problemas terem sido observados - como o que acontece quando a imagem contém caminhos que se cruzam -, o projeto ajudou a desenvolver alguns conceitos básicos de processamento de imagem e habilidades de programação.
