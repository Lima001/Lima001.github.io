Sobre Link para o cabeçalho

O projeto consiste em implementar elementos básicos necessários para a simulação de um robô diferencial em OpenGL (core). O objetivo, além de possibilitar o desenvolvimento de trabalhos futuros, é de aplicar de forma prática conceitos de computação gráfica na linguagem C++. Ao final, foi possível implementar um protótipo de simulação. Além disso, foi possível estudar de forma detalhada diferentes assuntos, sendo possível destacar a computação 3D, e o uso de curvas de Bézier. Juntamente disso, tópicos como otimização matemática e solução de raízes foram trabalhados para solução de problemas de renderização de curvas.

Conteúdos e tópicos estudados Link para o cabeçalho

  • Desenvolvimento usando C++ e OpenGL
  • Pipeline gráfico do OpenGL
  • Transformações
  • Curvas de Bézier e funções paramétricas
  • Renderização de curva de Bézier quadrática e cúbica
  • Otimização numérica
  • Método de Newton
  • Cinemática de um robô diferencial no espaço
  • Computação gráfica 3D
