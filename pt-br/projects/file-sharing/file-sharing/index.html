Protótipo de sistema de compartilhamento de arquivos P2P: Enviando mensagens secretas por imagens

Informações básicas Link para o cabeçalho

  • Autores:
    • Estudante: Gabriel Eduardo Lima
    • Professor: Hylson Vescovi Netto

Sobre Link para o cabeçalho

O projeto consiste na implementação de uma rede para compartilhamento de mensagens “secretas” através de imagens esteganografadas com uma mensagem de texto critpografada. O protótipo de uma rede P2P foi implementado para o acesso e distribuição de arquivos. Para isso foram implementados em Python um servidor de páginas HTML, e um protótipo de rede P2P para operarar como servidor de arquivos. Apesar de alguns problemas, foi possível observar como resultado o armazenamento, compartalhimaneto e recuperação das mensagens ocultadas em imagens.
